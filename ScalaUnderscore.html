Scala下划线用法

scala下划线有很多作用，可以作为通配符、类成员默认值、可变参数、类型通配符、默认匹配、 匹配集合元素、访问Tuple元素等功能，下面详细介绍。
    1、通配符import java.util._
    2、类成员默认值var name: String = _
    3、可变参数  //args: String*可变参数
  def echo(args: String*) = {
    for (arg <- args) println(arg)
  }

  echo(Array("aaa","bbb"): _*)

    4、类型通配符
      def echo(args: List[_]) = {
    args.foreach(arg => {
      println(arg)
    })
  }
  echo(List("aaa", "bbb", "ccc"))

    5、默认匹配
        val str=""
    str match {
      case "1" => print("111111")
      case _ => print("other")
    }
    //输出other
    6、匹配集合元素
      def parse(x: Any): Unit = {
    x match {
      case list: List[_] => "匹配List"
      case List(0, _, _) => println("匹配以0开头，长度为三的列表")
      case List(0, _*) => println("匹配以0开头，长度任意的列表")
      case list @ List(1, _*) => "list : $list"
      case Array(_*) => println("一个数组")
      case Vector(_*) => println("一个向量")
      case (a,b,c) => println(s"三元素元组")
      case (a,b,c,_) => println("四元素元组")
      case _ => print("other")
    }
  }
    7、Scala特有语法
    访问Tuple元素：    val t = (1, 2, 3, "65")
    println(t._1, t._2, t._3)
    简写函数:    val f1 = (_: Int) + (_: Int)
    //等价于
    val f2 = (x: Int, y: Int) => x + y

    list.foreach(println(_))
    //等价于
    list.foreach(e => println(e))

    list.filter(x => x > 0)
    //等价于
    list.filter(_ > 0)
    将方法转换成函数:
        val p = print _
    p("1")
