Gateone部署

GateOne 是一款使用 HTML5 技术编写的网页版 SSH 终端模拟器。基于现代的 HTML5 技术，无需任何浏览器插件、支持多个 SSH 进程、可以嵌入到其他任意应用程序中、支持使用 JavaScript，Python 甚至纯 CSS 编写的插件、支持 SSH 进程副本，打开多个进程而无需重复输入密码、 支持各种服务器端的日志功能，支持 Keberos-based 单点登录甚至活动目录、 支持操作日志记录，具有操作记录回放功能。

sudo yum install -y python-devel gcc dtach python-pip python-imaging python-kerberos
sudo yum install -y git pam-devel pam PyPAM
git clone https://github.com/tiwe-de/libpam-pwdfile.git
make && make install
ln /lib/security/pam_pwdfile.so /lib64/security/pam_pwdfile.so
htpasswd -c -d /etc/gateone/passwd newland
修改配置文件：sudo vi /opt/gateone/server.conf：		pam_service = "gateone"	auth = "pam"
vi /etc/pam.d/gateone(这个文件里用到了pam_pwdfile.so,和pam_permit.so，需要下载源码编译)
#%PAM-1.0
# Login using a htpasswd file
#@include common-sessionauth
#required pam_pwdfile.so pwdfile=/etc/gateone/passwd
#required pam_permit.so
auth            required        pam_pwdfile.so pwdfile=/etc/gateone/passwd
account         required        pam_permit.so
session         required        pam_permit.so
password        required        pam_deny.so

sudo service gateone restart
