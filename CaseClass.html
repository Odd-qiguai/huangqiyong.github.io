Scala的 case class学习

case class 样例类，适用于不可变数据，经常被用于匹配模式，实例化时不需要new，编译时会自动生成对象。
将类定义为case类会生成许多模板代码,如
case class Person (age: Int, name: String)
Person.scala在编译后会产生2个class，如图：

1、初始化的时候可以不用new，也可以加上，但是普通类一定需要加new；
    因为编译器给scala自动添加了一个伴生对象Person$，实现了apply方法如下：
    
    apply默认new了一个对象，所以我们创建一个实例的时候不需要使用关键字new,可以直接通过Person(age,name)来得到一个实例对象
2、默认实现了equals 和hashCode和toString；
    
3、默认是可以序列化的，也就是实现了Serializable ；
    
5、自动从scala.Product中继承一些函数(如上图);
6、case class构造函数的参数是public级别的，我们可以直接访问；
    
7、支持模式匹配；
