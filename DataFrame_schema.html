Spark中DataFrame的schema使用

Schema是DataFrame中的数据结构信息

算法要求：给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

for循环遍历2次

class Solution {
    public int[] twoSum(int[] nums, int target) {
        int[] result = new int[2];
        for (int i = 0; i < nums.length; i++) {
            for (int j = i + 1; j < nums.length; j++) {
                if (nums[j] == target - nums[i]) {
                    result[0] = i;
                    result[1] = j;
                    return result;
                }
            }
        }
        return result;
    }
}

使用哈希表遍历一次

使用HashMap，时间复杂度：O(n)，空间复杂度：O(n)
执行用时：2 ms， 内存消耗：39.8 MB
可模仿for循环遍历两次，但是参考其他答案，发现边迭代边插入，可以省一次遍历，提高效率。

class Solution {
    public int[] twoSum(int[] nums, int target) {
        Map<Integer , Integer> map = new HashMap<Integer , Integer>();
        int[] result = new int[2];
        for (int i = 0; i < nums.length; i++) {
            int rs = target - nums[i];
            if (map.containsKey(rs)) {
                result[0] = map.get(rs);
                result[1] = i;
                return result;
            }
            map.put(nums[i], i);
        }
        return result;
    }
}
