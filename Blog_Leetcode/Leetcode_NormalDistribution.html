正态分布随机数

一般我们经常使用的随机数函数 Math.random() 产生的是服从均匀分布的随机数，能够模拟等概率出现的情况，但现实生活中更多的随机现象是符合正态分布的，例如20岁成年人的体重分布等。 

方差越大，钟形越扁平，方差越小越陡:

下面利用nextGaussian()方法用于获取下一个伪高斯(“正常地”)分布的均值为0.0，标准差为1.0从此随机数生成器的序列的double值。

            
            import java.util.Random
/** *
    * 正态分布算法2
    *
    * @param mean    均值
    * @param std_dev 方差
    * @return 正态分布随机数
    */
  def getND2(mean: Double, std_dev: Double): Double = {
    val randomno = new Random
    Math.sqrt(std_dev) * randomno.nextGaussian + mean
  }
