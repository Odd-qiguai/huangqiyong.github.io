回文数

算法要求：判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

小于0有负号，不是回文数，10的倍数必定0结尾，不是回文数。优先排除。

剩余数字，从后往前取直到中间，如果位数为奇数，忽略中间值对比。数值和前面一样。为回文数。
如：1645461，拆解后revernum=1645，x=164， 1654/10取整等于x，为回文数。
代码如下：

class Solution {
    public boolean isPalindrome(int x) {
        boolean rs = false;
        if (x < 0 || (x != 0 && x % 10 == 0)) {
            rs = false;
        } else if (x > 11) {
            int revernum = 0;
            while (revernum < x) {
                revernum = revernum * 10 + x % 10;
                x = x / 10;
            }
            if (x == revernum || x == revernum / 10) {
                rs = true;
            }
        } else {
            rs = true;
        }
        return rs;
    }
}
